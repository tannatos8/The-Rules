# The Rules!

Este proyecto esta enmarcado por la materia Bases de Datos de la facultad de Ingeniería de Sistemas e Informática de la **Universidad Pontificia Bolivariana Seccional Bucaramanga**, donde se busca la realización de las reglas del negocio de las bases de datos, las cuales se realizaban de una forma no muy eficiente, con este se buscó darle solución mediante este proyecto con medio de un registro de usuario que tenga los proyectos y sus respectivas reglas de negocio en el cual se pueden filtrar por nombre del proyecto y que muestre la reglas del negocio.


# Configuración

StackEdit stores your files in your browser, which means all your files are automatically saved locally and are accessible **offline!**

## Create files and folders

Se inicializa con la importación de la base de datos en Mysql Workbench y la descarga o clonación del proyecto, dentro del proyecto en la carpeta **Connection** dentro de la cual esta una clase llamada **MysqlConnection.cs** en la cual se tiene que cambiar el usuario y contraseña del usuario de la base de datos, posteriormente ejecutar el programa.
